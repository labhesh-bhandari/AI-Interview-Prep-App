# AI Interview Prep Platform

A full-stack AI-powered interview preparation platform that helps users generate role-based interview questions, explore detailed answers, learn concepts in-depth, and manage practice sessions — all in one place.

---

## Features

### Authentication
- User registration & login with JWT
- Profile management with image upload

### AI Interview Preparation
- Generate interview questions based on role, experience & topics
- Get beginner-friendly answers with code examples
- “Learn More” feature for deep concept explanations

### Session Management
- Create & manage interview sessions
- Store questions and answers
- Pin important questions
- Delete sessions

### UI/UX
- Clean and responsive interface
- Modal & drawer components
- Loading states and toast notifications
