# AI Interview Prep – Frontend

A modern React-based frontend for an AI-powered interview preparation platform that helps users generate, practice, and manage interview questions tailored to their role, experience and Topics to Focus.

---

## Features

- Authentication (Login / Signup with profile photo)
- User profile management
- Dashboard with interview sessions
- AI-generated interview questions
- Pin / Unpin important questions
- Expandable explanations to get a gist of the Answer
- “Learn More” deep-dive responses
- Session creation & deletion
- Clean UI using TailwindCSS
- Fast and responsive experience

---

## Tech Stack

- React
- React Router Dom
- Axios
- Tailwind CSS
- React Markdown
- React Icons
- React Syntax Highlighter
- React Hot Toast

---

## Core Components

### User Context
Handles:
- user state
- authentication
- token persistence

### Dashboard
Displays:
- All sessions
- Summary cards
- Delete functionality

### AI Response Preview
- Markdown rendering
- Code highlighting
- Copy-to-clipboard support

### Create Session
- Generates AI questions
- Stores session in backend
